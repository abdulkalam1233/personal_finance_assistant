Build an AI assistant that:

Understands user financial data (bank statements, SMS, UPI, PDFs)
Categorizes expenses intelligently
Provides insights, alerts, and recommendations
Answers questions like:
“Where did I overspend last month?”
“How can I save more tax?”
“Can I afford a ₹15L car?”


### MVP
- Features
  - Auth
  - Upload statement PDFs
  - Parse PDFs using LLMs and create embeddings
  - Classify the statements and store the transactions
  - Create dashboard for monthwise/year transactions
